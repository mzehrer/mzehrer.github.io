This includes applications and also development stuff. If you want to suggest a link, leave a comment at the bottom of this page.

Runtime

  • Gtk+ for Win32 Runtime and development versions of GTK+ and other assorted libraries.

Office

  • AbiWord AbiWord is a free word processing program similar to Microsoft® Word. It is suitable for a wide variety of word processing tasks.
  • Planner Planner is a project managment tool for the Gnome desktop, for planning, scheduling and tracking projects.

Graphics

  • The Gimp Installers for the poular GNU Image Manipulation Program and the GTK+ 2 runtime.
  • Sodipodi There is always a Win32 version of this vector-based drawing program in the download section.
  • Inkscape Inkscape is an Open Source vector graphics editor, with capabilities similar to Illustrator, Freehand, CorelDraw, or Xara X using the W3C standard Scalable Vector Graphics (SVG) file format.
  • autopano-sift Panorama images are wide-angle images that amaze people: you often feel being inside the scene when watching a good panorama image. Creating such images is easy and everybody with a digital camera and a bit of patience can do it. Autopano-SIFT is there to make the creation of panorama images more fun.

Internet

  • Gaim A Win32 Port of the instant messaging client gaim.
  • Pan newsreader Win32 binaries for the Pan newsreader.
  • Sylpheed / Claws The Sylpheed / Claws mail client.
  • X-Chat XChat is an IRC (chat) program

Utils

  • Gkrellm A Win32 port of the System Monitor Gkrellm and addtional plugins.
  • Ettercap Ettercap is a suite for man in the middle attacks on LAN. It features sniffing of live connections, content filtering on the fly and many other interesting tricks.
  • Ethereal Ethereal® is used by network professionals around the world for troubleshooting, analysis, software and protocol development, and education.

Development

  • Glade/GTK+ for Windows There is a Win32 version of the GTK+ GUI builder and also versions of the gtk+ runtime.
  • Mono and GTK# A Win32 setup of the Mono runtime and developer tools, including the gtk# bindings, is avalable from the download page.
  • gtk2-perl Perl Win32 bindings for Gtk+ 2 and various related libraries.
  • PyGTK Python GTK+ bindings for Win32.
  • Gtk# for Microsoft .NET Framework SDK A Gtk# Software Development Kit Win32 Installer for the Microsoft .NET Framework 1.1 SDK including Glade and Gtk# application Project Templates for C# projects.
