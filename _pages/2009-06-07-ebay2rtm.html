ebay2rtm is a Greasemonkey script that adds a link to Ebay auction pages from which you can add reminders for these auctions to Remember The Milk.

At the moment this script is only tested on the german Ebay pages.

Installation + Source:
Install ebay2rtm script. development takes place at Github.
