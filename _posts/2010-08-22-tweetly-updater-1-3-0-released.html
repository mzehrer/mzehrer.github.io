There is a new release 1.3.0 of my Wordpress plugin Tweetly Updater. It includes the following changes:

  • Switch to Twitters new OAuth authentication protocol. I'm sorry, I was a bit late on this.
  • Tweeting of page updates is now optional.
  • You can add static hashtags to your tweets.
I hope you will have fun with the update, please continue reporting bugs or ideas via the issue tracker.
