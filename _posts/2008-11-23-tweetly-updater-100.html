I released a Wordpress plugin that updates Twitter when you create or edit a blog entry, it uses the bit.ly service for short urls.

The first release 1.0.0 has the basic features of updating twitter and url shortening, in future releases I will add support for the additional info that bit.ly stores (Example) for submitted URLs.
